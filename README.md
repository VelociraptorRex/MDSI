# MDSI — Español esencial

Содержательная версия 1.2 системы минимально достаточного изучения языка. Пакет `es-419` построен как спокойная связанная энциклопедия: 500 лексических единиц, 15 грамматических тем, 32 паттерна, 8 диалогов, банк из 2 187 упражнений и готовые карточки Anki, Quizlet и Полиглот используют общую модель данных и общие компоненты.

Практика поддерживает счёт по каждому уровню, выбор от 10 до 100 заданий на узнавание, управление ответами клавишами 1–4, автоматический переход после правильного ответа, сборку фразы из слов и ручной ввод, а также честную самопроверку свободных ситуационных реплик. На первых трёх уровнях правильный испанский ответ однократно озвучивается предпочтительно женским испанским голосом браузера; общий переключатель позволяет мгновенно отключить и снова включить звук.

Лексикон хранит полную модель спряжения для всех 128 глаголов, но показывает компактное учебное ядро: четыре времени Indicativo и Imperativo. Карточка открывается временно по наведению или закрепляется щелчком; целевая форма во всех 500 естественных примерах выделяется красным, а отдельная кнопка озвучивает лемму.

## Запуск

```bash
npm install
npm run dev
```

Производственная сборка:

```bash
npm run build
```

Готовые статические страницы появляются в `dist/`.

Anki и Quizlet пересобираются из лексикона, банка фраз и паттернов:

```bash
npm run generate:cards
```

Результаты `.apkg` для Anki и `.txt` с табуляцией для Quizlet сохраняются в `public/downloads/`. Таблица Полиглота `.xlsx` строится из тех же 1 044 нормализованных строк: семь совместимых колонок, без обратных дублей, с полностью пустым столбцом «Выучено». Все три файла автоматически попадают в статическую сборку сайта.

## Локальный пакет для macOS

```bash
npm run package:local
```

Команда создаёт папку `local-package/MDSI-local` с готовой сборкой и файлом `Открыть MDSI.command`. Он запускает локальный сервер и открывает сайт в браузере. Непосредственное открытие HTML через `file://` не поддерживается браузерами для приложений с JavaScript-модулями.

## Данные

Исходный файл лексического ядра находится в `content/es-419/lexico_es_500_v01.xlsx`. Команда `npm run import:lexicon` преобразует его в машинный JSON для интерфейса и проверяет наличие ровно 500 строк.

## Архитектура

- `src/components/` — общие компоненты: шапка, поиск, содержание, popover, легенда;
- `src/pages/` — полноценные страницы всех основных разделов;
- `src/styles/` — дизайн-токены, глобальные, компонентные и страничные стили;
- `src/data/` — машинные данные, включая автоматически генерируемые таблицы спряжения;
- `scripts/` — импорт данных и генерация статических HTML-маршрутов;
- `content/es-419/` — содержательный пакет языка, отделённый от интерфейсного ядра.

Цветовая семантика: красный — окончания, синий — формы и множественное число, зелёный — неправильные формы, фиолетовый — паттерны, серый — грамматические комментарии.

Светлая и тёмная темы используют одну систему токенов; выбор пользователя сохраняется локально в браузере.
